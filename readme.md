# Options Pricing Calculator
App generated by OpenAI Chat GPT with GPT-4 model

## About
Prices options using Black-Scholes Model

## Installation
### Windows
0. winget install python
1. `pip install flask`
2. `pip install scipy`
3. `python app.py`

### Mac
0. install homebrew
1. `pip3 install fask`
2. `pip3 install scipy`
3. `python3 app.py`
